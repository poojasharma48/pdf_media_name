/**
 * Installation
*/

* install module by following steps:
   unzip the pdf_media_name module and place in custom module
   
* Enable "pdf_media_name" module as usual.

* After installation , Go to media (media url) : https://your_domain_name/admin/content/media

* Then click on "Add media" link then this new page will https://your_domain_name/media/add

   - There click on "File pdf" link then this Page will open: https://your_domain_name.com/media/add/pdf_file

   - Then add the media name and Upload pdf file and click on save
   
Output:
 Then you will see when you view pdf media on frontend it will display media name instead of pdf name
